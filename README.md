# 📱 Social Media Hub

A simple and clean social media links website that displays our company's official social media profiles.

## 🔗 Platforms Included

- 🎥 **YouTube** – Videos, tutorials and product launches
- 💼 **LinkedIn** – Professional updates and job openings  
- 📸 **Instagram** – Photos, reels and behind-the-scenes
- 📘 **Facebook** – Events, announcements and community

## 🚀 How to Use

1. Open `index.html` in any web browser
2. Click on any social media card
3. You will be redirected to our official page

## 🛠️ Built With

- HTML
- CSS
- JavaScript
